## 一、GSE242252 — HZ疾病全血转录组

### 数据文件

| 文件名 | 大小 | 内容 |
|--------|------|------|
| **raw_counts.csv** | 7.9M | 原始count矩阵：35,859基因 × 80样本 |
| **filtered_counts.csv** | 5.3M | 过滤后count矩阵：20,786基因 × 80样本（≥10 counts在≥3个样本） |
| **sample_metadata.csv** | 8K | 80个样本的元数据（condition, patient_id, timepoint） |
| **deseq2_metadata.csv** | 8K | DESeq2用的元数据（含group列：Control_acute, Herpes_Zoster_acute等） |
| **DE_HZ_acute_vs_convalescent.csv** | 1.9M | **完整DESeq2结果**：15,227个基因的baseMean, log2FoldChange, lfcSE, stat, pvalue, padj, gene_id |
| **DE_HZ_annotated.csv** | 2.2M | 上表+ensembl_id_clean和symbol两列（gene symbol来自mygene映射） |
| **HZ_log2FC_for_comparison.csv** | 1.1M | **用于跨数据集比较的精简版**：gene_id, ensembl_id, symbol, log2FC_HZ, padj_HZ |

**DEG统计**：549个上调，769个下调（FDR<0.05），其中|LFC|>0.5的上调基因105个、下调基因60个。

### 图片

| 文件名 | 内容 |
|--------|------|
| **Volcano_HZ_acute_vs_convalescent.pdf** | **火山图** — X轴=log2FC（急性期vs恢复期）、Y轴=-log10(padj)。红色=上调(FDR<0.05 & LFC>1)，蓝色=下调，标注top 20基因名。包含统计数字（549 up, 769 down FDR<0.05） |
| **PCA_HZ_allgenes_PC1vsPC2.pdf** | **PCA图（全基因集）** — 急性期(红) vs 恢复期(蓝)样本在PC1/PC2空间中的分布。用于展示性染色体基因对PC1的潜在影响（方法学质控说明） |

## 三、通路富集与功能模块

### 数据文件

| 文件名 | 内容 |
|--------|------|
| **GO_BP_upregulated.csv** | **HZ上调基因的GO BP富集**（Enrichr, GO_Biological_Process_2023）。672个terms。列：Gene_set, Term, Overlap, P-value, Adjusted P-value, Odds Ratio, Combined Score, Genes（;分隔） |
| **GO_BP_downregulated.csv** | HZ下调基因的GO BP富集。402个terms。格式同上 |
| **modules.json** | **功能模块定义**（JSON格式）。9个模块的name、基因数、具体基因列表和来源GO terms |
| **module_Type_I_IFN_Antiviral.txt** | Type I IFN & Antiviral Response：11个基因（ISG15, RSAD2, IFI44L, IFI27, MX1, IFIT5等） |
| **module_Cell_Cycle.txt** | Cell Cycle & Proliferation：6个基因（TOP2A, MKI67, PTTG1等） |
| **module_T_Cell_Activation.txt** | T Cell Activation：5个基因 |
| **module_B_Cell_Humoral.txt** | B Cell & Humoral Immunity：4个基因（MZB1, IGLC3等） |
| **module_Antigen_Presentation.txt** | Antigen Processing：1个基因 |
| **module_Signaling.txt** | Signaling Pathways：3个基因 |
| **module_Cell_Adhesion.txt** | Cell Adhesion：1个基因 |
| **module_HZ_Disease_All_Up.txt** | HZ Disease Signature (All Up)：102个基因（所有上调DEGs LFC>0.5, FDR<0.05） |
| **module_HZ_Disease_All_Down.txt** | HZ Disease Signature (All Down)：48个基因（所有下调DEGs LFC<-0.5, FDR<0.05） |

**9个功能模块定义**：
1. Type I IFN & Antiviral（11基因）— Defense Response To Virus GO term
2. Cell Cycle & Proliferation（6基因）— Mitotic Spindle Organization GO term
3. T Cell Activation（5基因）— Antigen Receptor Signaling GO term
4. B Cell & Humoral（4基因）— BCR Signaling GO term
5. Antigen Processing（1基因）
6. Signaling（3基因）
7. Cell Adhesion（1基因）
8. HZ Disease All Up（102基因）
9. HZ Disease All Down（48基因）
